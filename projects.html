Projects I've Worked On


House of Health

I created an interface for use by receptionists at health centers. It allows for the user (receptionist) to update patient records, create or cancel patient appointments, create or cancel patient procedures, and create patient's accounts. The interface is designed to allow the user to update an SQL database using the information they enter into each form.

Skills: HTML, CSS, JavaScript, PHP, SQL

Quake 4: The Palmod

I created a modification for the game Quake 4, which allows the user to catch five different "Pals" using your trusty "Palcatcher" and command them each to do unique attacks using your weaponry! This project allowed me to highlight my ability to work with an existing codebase and modify it to allow games to be played in different ways than intended.

Skills: C++
